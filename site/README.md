# GACE Scan Site

Static App Store support pages for GACE Scan.

## Pages

- `index.html` - marketing landing page
- `support.html` - App Store Support URL page
- `privacy.html` - App Store Privacy Policy URL page
- `terms.html` - Terms of Use

## App Store URLs

Once hosted, use:

- Support URL: `https://gacescan.com/support.html`
- Privacy Policy URL: `https://gacescan.com/privacy.html`
- Marketing URL: `https://gacescan.com/`

If you publish with a host that gives you a temporary domain, replace `gacescan.com` with that domain until the custom domain is connected.
